Explore the Meet application

The Meet application is a React application that uses the Google Calendar API to find and filter events based on the selected city and number of events. Meet is an application that has online/offline capabilities, allowing its users constant access to their desired information.

A number of technologies were used to develop this application, such as Cucumber AWS Lambda, Google Calendar API and Recharts. Testing tools like Jest and Vite ensured the reliability of this application and its functionality.
